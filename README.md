# Sentiment Analysis

## Project overview
This project builds a Machine Learning model to detect the IMDb movie reviews and classify them as positive or nrgative. The goal is to understand audience opinions from the text data.

## Technologies Used
-Python
-Pandas 
-NumPy
-NLTK
-Stopwards

## Model
A **MultinomialNB** was trained to clasiify reviews as Positive or Negative.

## Evualtion Metrics
-Accuracy
-Confusion Matrix
-Precision, recall, F1-Score

## Outcome
Accuracy: The model is correct about 86% of the time when predicting if a review is positive or negative.
Precision: When the model predicts a review as "negative":
It's right about 84% of the time. When a review is actually negative, it catches 88% of them. Precision: When the model predicts a review as "positive":
It's right about 87% of the time. When a review is actually positive, it catches 84% of them. F1-score: This number combines both precision and recall. It's a balanced measure of correctness
